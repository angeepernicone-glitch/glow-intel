The [skincare](/blog/how-to-fade-dark-spots-hyperpigmentation) routine order question gets more complicated as people add more products. The generic advice — "apply thinnest to thickest" — is a start, but it misses the ingredient interactions that actually determine whether your actives work or fight each other.

## The General Framework

Thinnest to thickest texture is a reasonable starting rule because:
- Thinner products (water-based serums) need direct [skin](/blog/skincare-routine-acne-prone-skin) contact to penetrate
- Heavier products (oils, creams) form films that would block lighter products applied after them

But texture is a proxy for what you really care about: ensuring each active reaches skin cells and doesn't get inactivated by something applied before or after it.

## The Morning Stack

**1. Cleanser** → **2. Toner/Prep step (optional)** → **3. Vitamin C serum** → **4. Niacinamide serum (optional)** → **5. Lightweight moisturizer** → **6. SPF**

The vitamin C [goes first](/blog/niacinamide-vs-vitamin-c-which-goes-first) in the morning for two reasons: it's more stable on clean skin, and it provides photoprotection that stacks with SPF. Apply to slightly damp skin.

The old rule about "never mix vitamin C and niacinamide" is outdated. That concern was based on old formulations. Modern stable vitamin C (L-ascorbic acid or vitamin C derivatives) can be layered with niacinamide without issues. If you're using an unstable vitamin C and seeing flushing, it's usually the oxidation of the vitamin C, not the niacinamide combination.

## The Evening Stack

**1. Cleanser** → **2. Exfoliant (2-3x/week only)** → **3. Targeted serums** → **4. Retinol** → **5. Moisturizer** → **6. Occlusive (optional)**

The key decision in the evening stack: when to place retinol. Most guidance says apply retinol before moisturizer, but for those starting out or with dry/sensitive skin, applying moisturizer first (the "retinol sandwich") reduces penetration and irritation while still delivering results over time.

## The Interactions That Actually Matter

**Vitamin C + AHAs/BHAs**: Both are acidic, so they're pH-compatible, but the combined exfoliation + vitamin C sensitivity can be too much for most skin. Don't use them at the same time. Vitamin C in the morning, acids in the evening.

**Retinol + AHAs/BHAs**: Don't use on the same night, especially when starting. Retinol is irritating enough on its own. If you need both, alternate: acid on Monday/Wednesday, retinol on Tuesday/Thursday. Once your skin is fully adapted (3+ months), some people tolerate both in the same evening routine.

**Retinol + Vitamin C**: Not recommended in the same application. Vitamin C (L-ascorbic acid) works best at pH 2.5-3.5, retinol at neutral pH. They don't inactivate each other but they're not optimal together. Morning for C, evening for retinol — simple.

**Niacinamide + Retinol**: Actually helpful together. Niacinamide is anti-inflammatory and can offset some of retinol's irritation. Apply niacinamide serum first, let it absorb, then retinol.

**Copper peptides + Vitamin C**: These can chelate (bind together), which can inactivate the vitamin C. Use at different times of day.

## Common Mistakes

**Using too many actives simultaneously when starting**: If you just added retinol to your routine, don't also start vitamin C, a new AHA, and peptides in the same week. One new active at a time. Wait 2-3 weeks before adding the next.

**Applying actives to wet skin** (when you shouldn't): Wet skin enhances absorption — which is useful for hyaluronic acid (apply to damp skin), but can increase irritation with retinol or strong acids. Pat skin mostly dry before applying retinol and high-percentage AHAs.

**Not waiting between steps**: You don't need to wait 10-20 minutes between every product (that advice comes from the tretinoin prescription world where waiting genuinely matters). For most actives, 30-60 seconds of absorption is enough before applying the next step.

## A Realistic Weekly Schedule

**Monday, Wednesday, Friday evenings**: Exfoliant (AHA or BHA) → serum → moisturizer. No retinol these nights.

**Tuesday, Thursday, Saturday evenings**: Niacinamide → retinol → moisturizer. No acid these nights.

**Sunday evenings**: Simple barrier repair night — just cleanser and a rich moisturizer. No actives.

**Every morning**: Vitamin C → moisturizer → SPF.

This schedule gives each active space to work without overcrowding or overlapping irritation.

## The Honest Bottom Line

More actives don't mean better skin. The best routines I've seen are consistent, not complex. If you can maintain four products daily — cleanser, vitamin C in the AM, retinol 3x per week at night, moisturizer, SPF — you'll outperform someone cycling through ten different serums monthly.

Consistency over time beats complexity every time.
